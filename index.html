Github+Hexo博客搭建教程

本文环境为windows

Github

为什么用Github

1、GitHub Pages本用于介绍托管在GitHub的项目，不过，由于他的空间免费稳定，用来搭建一个博客再好不过了
2、Github Pages有300M免费空间，资料自己管理，保存可靠。
3、Github Pages可以被认为是用户编写的、托管在github上的静态网页。

安装准备软件

1、Node.js
2、Git

Windows安装Node.js以及Git直接在网上下载可执行的安装版本，然后按照提示安装即可。

注册Github

1、如果没有Github账户则需要先注册一个Github账户，登录https://www.github.io注册账号。
2、完成注册后需要创建一个以{用户名}.github.io格式命名的仓库，用于存放我们的博客项目。注意这种类型的仓库仅能创建一个。
3、远程仓库创建完成后需要和本地Git建立连接。
