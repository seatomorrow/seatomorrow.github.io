Github+Hexo博客搭建教程

  • Github
  • Hexo
  • 个人博客
奇淫技巧

本文的操作环境为Windows 7。

Github

为什么用Github

  1. GitHub Pages本用于介绍托管在GitHub的项目，不过，由于他的空间免费稳定，用来搭建一个博客再好不过了
  2. Github Pages有300M免费空间，资料自己管理，保存可靠。
  3. Github Pages可以被认为是用户编写的、托管在github上的静态网页。

安装准备软件

  1. Node.js
  2. Git

Windows安装Node.js以及Git直接在网上下载可执行的安装版本，然后按照提示安装即可。

注册Github

  1. 如果没有Github账户则需要先注册一个Github账户，登录https://www.github.io注册账号。
  2. 完成注册后需要创建一个以{用户名}.github.io格式命名的仓库，用于存放我们的博客项目。注意这种类型的仓库仅能创建一个。

SSH Keys

远程仓库创建完成后需要和本地Git建立连接。这里是通过SSH Keys来实现的。

  1. 首先检查电脑上现有的ssh key。

3、连接Github账户。
登录Github，进入settings->SSH keys->New SSH key，Title可以随便写，把~/.ssh目录中文件id_rsa.pub的内容复制到Key，点击Add SSH key完成连接。

4、设置免SSH密码。
默认每次向Github push代码或者使用hexo deploy时都要输入Github账户名和密码。通过以下方法可以设置免密码登录。
进入%HOME%(默认c:/Users/xhm/)目录，新建一个名为”_netrc”的文件，文件中内容格式如下：
machine {git account name}.github.com
{your usernmae}
{your password}
重新打开git bash即可，无需再输入用户名和密码。
